# MinecraftJavaVersions

|  サーバー  |  JAVA  |
| ---- | ---- |
|  1.7.10 ~ 1.16.5  |  Java 8/11*  |
|  1.17 ~ 1.17.1  |  Java 16  |
|  1.18 ~  |  Java 17**  |

- *Java 8/11には互換性がある
- **少なくとも1.20までは Java 17を使用することが[開発者(md_5)により推奨される](https://www.spigotmc.org/threads/which-java-version-is-recommended-for-1-20.619359/)。
